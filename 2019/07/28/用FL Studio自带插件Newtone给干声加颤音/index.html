用 FL Studio自带插件Newtone给干声加颤音

  1. 1. 进入正题

在使用Melodyne调声的时候遇到了问题，Melodyne的Pitch工具仅支持对音符原有颤音曲线的幅度进行调整（包括颤音幅度和偏移），有很大局限。
查询后，发现水果的自带的Newtone竟附带了高级颤音编辑功能,以前都没发现哈哈哈。使用过后感觉不错，在这里写一下做下备忘。

进入正题

首先进入水果，打开Newtone，推拽素材进入Newtone。
导入后就可以看到一个个的音符了。
在目标音符上双击，进入高级编辑模式。
选择右上角剪刀工具内的Advanced edit vibrato选项。
调整蓝色竖线来改变颤音频率。
在音符左右侧按住鼠标左键移动分别调整音符左右侧的颤音音量。
非常方便，做好之后保存就完事了。
